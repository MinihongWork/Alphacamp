## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

安裝完畢後，繼續輸入：

   ```bash
   npm run start
   ```

若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```bash
   Listening on http://localhost:3000
   ```

若欲暫停使用

   ```bash
   ctrl + c
   ```
使用環境

    body-parser 1.19.2
    express 4.17.1
    express-handlebars 4.0.2
    method-override 3.0.0
    mongoose 6.2.7
